# 🌍 TVSphere

TVSphere is a modern IPTV application that allows users to discover and watch free-to-air TV channels and listen to FM/Radio stations from around the world. It provides a simple, fast, and user-friendly experience with channels organized by country.

## ✨ Features

- 📺 Watch live TV channels from multiple countries
- 📻 Listen to FM and radio stations
- 🌍 Browse channels by country
- 🔍 Search for channels
- 🌙 Light & Dark Theme
- 🌐 Multi-language support
- 📡 Automatic country detection
- ❤️ Save favorite channels
- ▶️ Built-in video player
- ⚡ Fast and responsive interface
